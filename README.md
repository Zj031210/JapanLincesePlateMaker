# 使用方法（本地使用）
- 安装Node.js,链接:https://nodejs.org/en/
- 直接下载源码，解压出来，找个空的文件夹放在里面
- 用管理员方式打开cmd（或Windows10/11的Windows终端也可以）
- npm install -g typescript（注意，运行前打开VPN）
- cd 文件夹路径（你解压到的那个）
- npm install
- npm run build
- npm run start
- 等它显示Complie Successful之后，打开浏览器
- 地址输入localhost:8080
